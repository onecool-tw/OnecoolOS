# Onecool OS v0.1.0 Alpha

Onecool OS v0.1.0 Alpha freezes the lightweight layered architecture for the
project foundation.

## Completed Modules

- Core Engine
- Configuration System
- Logging System
- Scheduler System
- Market Engine foundation
- Asset modules foundation
- Connector and Normalize layers
- Ledger foundation
- Valuation platform foundation
- Portfolio aggregation foundation
- Analytics foundation
- Service Layer foundation
- Dashboard foundation

## Architecture Summary

The Alpha architecture is organized as:

```text
External Sources
↓
Connector
↓
Normalize
↓
Assets
↓
Ledger
↓
Valuation
↓
Portfolio
↓
Analytics
↓
Services
↓
Dashboard
↓
OFAI
```

Each layer owns one source of truth. Services provide read-only access.
Dashboard is display-only. OFAI remains a future decision and recommendation
layer.

## Known Limitations

- No production web dashboard yet.
- No API server yet.
- No database persistence for user portfolio records yet.
- No real automation workflows yet.
- No OFAI decision engine yet.
- Market, valuation, and analytics workflows remain foundation-level.
- Live external integrations remain intentionally limited.
