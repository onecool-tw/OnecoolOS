# Onecool OS v0.4.0-beta

Version: `v0.4.0-beta`

Release Theme: Investment Performance Beta

Onecool OS v0.4.0-beta prepares the first Investment Performance Beta release.
It freezes the deterministic unrealized performance loop built on ADR-005:
Investment Performance and Asset Lifecycle.

This release is release preparation only. It does not create a Git tag yet;
tagging is deferred until final human approval.

## Release Summary

v0.4.0-beta extends Onecool OS from Collectible Radar market intelligence into
investment performance context. Existing holdings use opening cost basis, with
historical transaction backfill remaining optional. The release keeps
performance deterministic, replayable, auditable, and free of recommendations
or AI reasoning.

The release focuses on unrealized performance for collectible holdings. It
does not include FX conversion, realized gain/loss, IRR/XIRR, annualized
return, live APIs, prediction, or recommendation workflows.

## Closed Loop

```text
Assets
↓
Valuation
↓
Investment Performance
↓
Dashboard
↓
Daily Report
↓
Decision Queue
↓
OFAI Context
```

### Assets

Assets describe identity and source facts. Collectible records preserve PSA/BGS
card identity, cost basis source fields, acquisition date, and currency.

### Valuation

Valuation preserves immutable valuation observations. It provides the market
value input used by Investment Performance. Valuation does not calculate
performance.

### Investment Performance

Investment Performance calculates only:

- cost basis
- market value
- unrealized gain/loss
- unrealized gain percent
- holding days

It does not calculate FX, IRR/XIRR, realized gain/loss, annualized return,
source agreement, confidence, prediction, or recommendations.

### Dashboard

Dashboard displays performance snapshots through structured presentation
sections. It does not calculate performance.

### Daily Report

Daily Report assembles existing Dashboard performance output into report
sections. It does not calculate performance.

### Decision Queue

Decision Queue prioritizes performance review work only. Missing Cost Basis and
Missing Market Value are Critical. Insufficient Data and Currency Mismatch are
High. Missing Holding Date is Medium. Performance Review Only is Low.

Decision Queue does not recommend buy/sell actions.

### OFAI Context

OFAI Context prepares deterministic investment-performance context for future
AI reasoning. It does not call LLMs, predict prices, recommend actions, or
mutate source data.

## Upgrade Notes

- No database migration is required.
- No private user data should be committed.
- Existing Collectible Radar imports remain local-file based.
- Opening cost basis remains the accepted starting point for existing
  holdings.
- Cost basis remains in original source currency until a future FX Engine is
  introduced.
- Performance outputs are unrealized-only in this release.
